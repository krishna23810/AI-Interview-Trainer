# AI Interview Trainer Agent

This project is a capstone case study on building an AI-powered Interview Trainer using IBM Cloud and RAG (Retrieval-Augmented Generation).

## 🌟 Features
- Generates technical and behavioral interview questions
- Provides feedback, model answers, and improvement tips
- Built with IBM Granite, Agent Lab, and Watsonx tools

## 🛠 Technologies Used
- IBM Cloud Lite
- IBM Granite Foundation Model
- Agent Lab
- Retrieval-Augmented Generation (RAG)
